Didn't specify what the problem was or how tp go about fixing it, so I'm left with a non-functioning keyboard now. Could not figure out how to temporarily disable the touch pad. The down button (arrow) worked when it wanted to - which was not often. The display - while beautiful and sharp - gets awful glare on the screen if you're using it in any environment other than a dark room. I think this product is in need of some serious reworking, honestly.
